# Relativistic-Solar-System-Simulator
A project focused on modeling relativistic effects and the solar system using PyOpenGL
The project is a simulator that aspires to offer an accurate representation of the solar system and the phenomenon of relativity linked to the planets.
In order to fulfill its purpose, it uses PyOpenGL for 3D modelisation and optimizes the cost in time for representing the curvature of time and space within the solar system.

# What was the motivation behind it ?
Initially I found one of my old school projects, that consisted of creating a "loading screen" using pygame. I thought about expanding on it by turning it into 3D before deciding to
be more ambitious.
I thus found myself working on understanding PyOpenGL and how to use it. The project took several months, but I am happy with the result.
The goal then became to offer a simulator, that was both informative but visual expanding on the original premise.
The addition of the curvature of time and space was the most difficult thing to do, and I also had trouble handling coordinates and the camera.
Nevertheless, I believe I have achieved what I wanted to do with this project which is offering an informative interface, but also representation of relativity and informations on the planets of the solar system.

# Useful information

In order to use the simulator, follow these instructions :

-> Download the ZIP folder associated with the project on github.
-> Open it, and launch "Launcher.py".
-> Select your language.
-> You can now use the simulator, by pressing the button to launch it. 

In the simulator, press V to switch between ways to view the solar system.
